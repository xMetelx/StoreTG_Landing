Часто задаваемые вопросы

Это безопасно?

Да, безопасно. Взаимодействие между пользователем, сервисом Auriva и Telegram осуществляется с использованием https протокола, который поддерживает безопасный способ передачи данных, посредством шифрования. Пароли пользователя хранятся в зашифрованном виде. Информация о платежных картах недоступна сервису Auriva и нигде не cохраняется.

Для каких товаров подходит сервис Auriva?

Сервис подходит для размещения любых товаров. Чат-бот может быть создан как для крупных интернет-магазинов, так и для небольших бизнесов.

Сколько стоит?

В настоящий момент сервис предоставляется бесплатно. В будущем сервис останется бесплатным для чат-ботов с небольшим количеством оформленных заказов. Для остальных чат-ботов будет браться ежемесячная абонентская плата в зависимости от количества заказов.
